Share screen with Java Client

Alternatively, you can share your screen using the Java client. You will need to have Java 1.5 or above installed. When beginning to screen share using the Java client, the browser downloads a JNLP file in order to start Java using Java Web Start (JWS). You should allow the JNLP file to launch JWS if you are prompted. You can keep the Java Client window open during a meeting and simply click on Start and Stop buttons in the window to start/stop a sharing session. Select the sharing region size and move the sharing region box in the Java Client window to position your screen sharing region accordingly.

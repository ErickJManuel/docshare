Hosting

 

The Hosting panel allows you to create hosting service profiles to be used by a group.

 

  • Web Conference profile refers to the hosing of the meeting site and all web conferencing functions, except for Video Conference and Remote Control functions. The default profile is the server hosting the meeting site.
  • Video Conference profile refers to the Webcam and voice over IP functions. The hosting profile is optional. If it is not set for a group, the Webcam function is disabled for all group members.
  • Teleconference profile refers to the teleconference server for recording and controlling an audio conference. The hosting profile is optional. If it is not set for a group, the recording function and audio controls functions are disabled for all group members.
  • Remote Control profile refers to the hosting of a remote control server that allows an attendee to control a presenter’s desktop during screen sharing. The hosting profile is optional. If it is not set for a group, the Remote Control function will be disabled for all group members.
  • Conversion Server profile refers to the hosting server for converting PowerPoint or other documents to a Flash format for adding content to the online library. The hosting profile is optional. If it is not set for a group, the Document Converter option is disabled for all group members.

 

 

After you have created a hosting account, you must assign it to a group for the account to be used. Click on Administration/Groups and select a group to assign the hosting account to.
